# Historia y Utilidad de Git

Git fue creado en **2005** por **Linus Torvalds**.

## Origen

Fue desarrollado para reemplazar el sistema de control de versiones que usaba el proyecto Linux, buscando una solución que fuera **rápida** y **distribuida** (sin servidor central).

## Propósito Actual

Hoy, Git sirve para:

1.  **Rastrear el Historial:** Guardar un registro de todos los cambios.
2.  **Colaboración:** Facilitar el trabajo simultáneo de equipos grandes.
3.  **Ramas:** Permitir el desarrollo aislado de nuevas funcionalidades.
