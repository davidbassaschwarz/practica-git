# Órdenes Principales de Git

* **`git init`** - Inicializa un repositorio nuevo.
    * **Ejemplo:** `git init`

* **`git status`** -  Muestra los cambios pendientes.
    * **Ejemplo:** `git status`

* **`git add <archivo>`** - Añade ficheros al área de preparación (índex).
    * **Ejemplo:** `git add .`

* **`git commit -m "mensaje"`** - Registra los cambios en el historial local.
    * **Ejemplo:** `git commit -m "feat: Finalizar tarea"`

* **`git branch <nombre>`** - Crea una rama nueva.
    * **Ejemplo:** `git branch nueva-rama`

* **`git checkout <nombre>`** - Cambia de rama.
    * **Ejemplo:** `git checkout main`

* **`git merge <rama>`** - Fusiona los cambios de una rama.
    * **Ejemplo:** `git merge historia-git`

* **`git push <remoto> <rama>`** - Sube los cambios al repositorio remoto (GitHub).
    * **Ejemplo:** `git push origin main`

* **`git pull`** - Descarga cambios del repositorio remoto al repositorio local.
    * **Ejemplo:** `git pull origin main`
